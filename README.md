# 快速必应搜索插件

一个简洁的浏览器新标签页插件，提供快速的必应搜索功能和美观的背景。

## 功能特性

- 🎨 固定使用 `cat.jpg` 作为背景
- 🔍 集成必应搜索功能
- ⚡ 快速加载
- 📱 响应式设计，适配各种屏幕尺寸
- 🖼️ 精美的图标

## 安装方法

### Chrome 浏览器

1. 下载本项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹
6. 完成安装，打开新标签页即可使用

### Edge 浏览器

1. 下载本项目到本地
2. 打开 Edge 浏览器，进入 `edge://extensions/`
3. 开启左侧的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹
6. 完成安装，打开新标签页即可使用

## 使用说明

1. 打开新标签页，即可看到带有 `cat.jpg` 背景的搜索页面
2. 在搜索框中输入关键词，按回车使用必应搜索
3. 输入网址（如 `https://www.example.com`），按回车直接访问该网站

## 自定义

如果需要修改背景图片：
1. 替换项目中的 `cat.jpg` 文件
2. 确保新图片命名为 `cat.jpg`
3. 重新加载扩展程序
